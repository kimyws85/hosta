JavaScript

웹에 생명을 불어넣는 프로그래밍 언어

JavaScript란?

JavaScript는 웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어입니다. 사용자 인터랙션, 데이터 처리, 애니메이션 등 다양한 기능을 구현할 수 있습니다.

주요 특징

🔄

동적 콘텐츠

실시간으로 콘텐츠를 업데이트하고 변경합니다

🖱️

이벤트 처리

클릭, 스크롤 등 사용자 상호작용을 처리합니다

🌐

API 통신

서버와 데이터를 주고받아 동적 웹을 구현합니다

⚙️

비동기 처리

Promise와 async/await로 비동기 작업을 처리합니다

기본 문법

// 변수 선언
const name = 'SBS Web';
let count = 0;

// 함수 정의
function greet(name) {
    return `안녕하세요, ${name}님!`;
}

// 화살표 함수
const add = (a, b) => a + b;

// DOM 조작
document.querySelector('.btn').addEventListener('click', function() {
    console.log('버튼이 클릭되었습니다!');
});

// 배열 메서드
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(num => num * 2);

// 비동기 처리
async function fetchData() {
    try {
        const response = await fetch('api/data');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error:', error);
    }
}
