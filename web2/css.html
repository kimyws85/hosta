CSS3

웹을 아름답게 꾸미는 스타일 시트

CSS란?

CSS(Cascading Style Sheets)는 HTML 문서의 스타일을 정의하는 언어입니다. 색상, 레이아웃, 폰트 등 웹 페이지의 시각적 표현을 담당합니다.

주요 특징

🎨

스타일링

색상, 폰트, 간격 등을 자유롭게 조절합니다

📐

레이아웃

Flexbox와 Grid로 현대적인 레이아웃 구성

✨

애니메이션

트랜지션과 키프레임으로 동적 효과 구현

📱

반응형

미디어 쿼리로 모든 화면 크기 대응

기본 문법

/* 선택자와 속성 */
.container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 20px;
}

/* Flexbox 레이아웃 */
.flex-container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    gap: 20px;
}

/* 애니메이션 */
.fade-in {
    animation: fadeIn 1s ease-in;
}

@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}

/* 미디어 쿼리 */
@media (max-width: 768px) {
    .container {
        padding: 10px;
    }
}

레이아웃 방법

Flexbox

1차원 레이아웃 시스템으로 행 또는 열 방향으로 아이템을 배치합니다.

display: flex;

Grid

2차원 레이아웃 시스템으로 행과 열을 동시에 제어합니다.

display: grid;

Position

요소의 위치를 절대적 또는 상대적으로 지정합니다.

position: absolute;

박스 모델

Margin
Border

모범 사례

  • 클래스 네이밍은 의미있고 일관성 있게 작성합니다
  • CSS 변수를 활용하여 재사용성을 높입니다
  • 미디어 쿼리로 반응형 디자인을 구현합니다
  • 성능을 위해 애니메이션은 transform과 opacity를 사용합니다
  • 브라우저 호환성을 고려하여 작성합니다
