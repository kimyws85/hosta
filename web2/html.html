HTML5

웹의 구조를 만드는 마크업 언어

HTML이란?

HTML(HyperText Markup Language)은 웹 페이지의 구조를 정의하는 마크업 언어입니다. 텍스트, 이미지, 링크 등 다양한 콘텐츠를 웹 브라우저에 표시하기 위한 기본 언어입니다.

주요 특징

🏗️

구조화

웹 페이지의 논리적 구조를 명확하게 정의합니다

♿

접근성

시맨틱 태그로 모든 사용자가 접근 가능합니다

🔗

하이퍼링크

페이지 간 연결을 통해 정보를 탐색합니다

📱

멀티미디어

이미지, 비디오, 오디오를 쉽게 삽입합니다

기본 구조

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>페이지 제목</title>
</head>
<body>
    <header>
        <h1>환영합니다</h1>
    </header>

    <main>
        <section>
            <h2>섹션 제목</h2>
            <p>내용을 작성합니다.</p>
        </section>
    </main>

    <footer>
        <p>© 2026</p>
    </footer>
</body>
</html>

모범 사례

  • 제목 태그(h1-h6)를 순차적으로 사용합니다
  • form 요소에는 label을 연결합니다
