## 📚 콘텐츠 구성


**Part 8. 자주 쓰는 패턴** 💡
- 버튼 패턴 (Primary/Outline/Disabled 버튼 코드)
- 카드 패턴 (기본 카드/호버 효과/이미지 카드)
- 폼 패턴 (Input/Label/Textarea 스타일)
- 레이아웃 패턴 (Header/Hero/Container/Grid 레이아웃)
- 유틸리티 조합 (실전 코드 스니펫)

**특징**
- 🎯 플랫 레이아웃: 모든 내용이 한 번에 보이는 구조 (접히지 않음)
- 🔍 실시간 검색: 클래스명/CSS 속성으로 즉시 검색 (테이블 + 패턴 필터링)
- 📊 표 형식: 종합적인 레퍼런스 테이블로 빠른 참조
- 💻 코드 스니펫: 복사해서 바로 쓸 수 있는 실전 패턴
- 🎨 색상 견본: 컬러 팔레트 시각적 확인
- 📱 반응형: 모바일/데스크탑 최적화

## 🎨 주요 기능

### 인터랙티브 UI
- **토픽 카드 확장/축소**: 각 주제를 클릭하여 세부 내용 확인
- **카테고리 네비게이션**: 파트별 빠른 이동 (스크롤 기반 활성화)
- **읽기 진행률 표시**: 상단 프로그레스 바로 학습 진행도 확인
- **스크롤 탑 버튼**: 400px 이상 스크롤 시 표시

### 반응형 디자인
- 데스크탑, 태블릿, 모바일 모든 화면 크기 지원
- 768px 이하에서 레이아웃 최적화

### 접근성
- 키보드 탐색 지원 (Enter/Space로 토픽 열기)
- ARIA 역할 속성 적용
- 시맨틱 HTML 구조

### 테마 시스템


## 🛠 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: CSS 변수, Flexbox, Grid
- **JavaScript**: Vanilla JS (프레임워크 없음)
- **폰트**: Noto Sans KR (Google Fonts)
- **아이콘**: Font Awesome 6.5.1
